# デザイン仕様

## スキルツリー(中央幹レイアウト)

- 画面下部のSTART円から中央の縦幹が上へ伸び、マイルストーンのピルが幹上に並ぶ
- スキルカードは幹から左右交互に枝分かれし、角丸の白カード(フラット・淡色)で表示する
- マイルストーンごとにパステルのアクセントカラーを循環使用: `#60a5fa` `#f472b6` `#34d399` `#fbbf24` `#a78bfa` `#2dd4bf`
- 葉(`leaves`)はスキルカードのすぐ外側へ緑の小ピル(🍃)としてファン状に枝分かれさせる
- 関連技術(`related`)は葉のさらに外側へアクセント色の小ピルとしてファン状に枝分かれさせる
- 背景は近白(`#f7f9fc`)のフラット。幹はグレー(`#c8d3dc`)の太い直線、枝はアクセント色のベジェ曲線
- 前提関係(`prerequisite_ids`)のエッジは描画しない(幹を横切るため)。並び順とロック表示・詳細パネルで依存を伝える。データと解放ロジックはそのまま

## ステータス表現

- done: アクセント色の塗り+チェック、in_progress: アクセント色のリング、locked: グレースケール+🔒、隠しスキルはアンバー+★
- マイルストーン: current はアクセント塗り、completed はアクセント枠、それ以外はグレー枠

## 共通

- デスクトップは右側詳細パネル(学習ステップ=葉と関連技術の一覧も表示)、モバイルは全幅パネル
- ローディングは処理内容を短い文で表示し、API障害時は固定デモへ自動フォールバック
