# デザイン仕様

画面構成の詳細とスクリーンショットは docs/ui-structure.md を参照。

## スキルツリー(全体ビュー: 中央幹レイアウト × コスミックガラス)

- ダークな宇宙背景(深紺グラデーション+星のドット+ネビュラの光彩)に、ガラス質(半透明+blur+発光)のノードを配置
- 画面下部のSTART円から中央の縦幹が上へ伸び、マイルストーンのガラスピルが幹上に並ぶ
- スキル(枝)は円形のガラスバブル+外側ラベルとして幹から左右交互に枝分かれ
- マイルストーンごとにネオンアクセントを循環使用: `#67e8f9` `#f0abfc` `#6ee7b7` `#fcd34d` `#a5b4fc` `#5eead4`
- **葉(`leaves`)は全体ビューでは小さな緑の芽(ドット)** としてバブルの外側に表示(ラベルなし=重なり防止)
- 関連技術(`related`)は芽の外側に小さなガラスピルとして枝分かれ
- ヘッダー下に総合達成率バッジ(doneノード数/全ノード数)
- 前提関係(`prerequisite_ids`)のエッジは描画しない。並び順とロック表示・詳細パネルで依存を伝える

## フォーカスビュー(要素ピックアップ)

- スキルバブルをクリックすると、そのノードを中心に**葉(ラベル付きピルに展開)と関連技術が円形に引き寄せられる**
- 他の要素は不透明度0.07までフェードし、カメラが `fitView(duration)` でズームイン
- ノードの移動は CSS transition(transform .75s)で滑らかに=引き寄せエフェクト
- 中心から放射状の直線エッジ(葉=緑 / 関連技術=アクセント色)
- 背景クリックで全体ビューへ戻る

## ステータス表現

- done: アクセント塗りバブル+✓発光、in_progress: パルス発光リング、locked: 減光+グレースケール+🔒、隠しスキルはアンバー+★
- マイルストーン: current はアクセント塗り、completed はアクセント枠、それ以外はガラス枠
- 葉: todo=緑枠 / done=緑塗り

## 共通

- デスクトップは右側詳細パネル(学習ステップ=葉と関連技術の一覧、証明日時も表示)、モバイルは全幅パネル
- ヘッダーの「振り返り」からモーダルを開き、AIによる自動整理(summarize-activity)と挑戦履歴を表示
- ローディングは処理内容を短い文で表示し、API障害時は固定デモ/テンプレートへ自動フォールバック
